PROJECT TITLE: Printing App

VERSION or DATE: 16/11/2021

AUTHORS: Ivan Potgieter

USER INSTRUCTIONS:
When the program launches, you will be prompted to open a file. Open the "Colours.txt" text file. The colours contained in the text file will be loaded into the listbox. Enter your surname, student number and the number of pages you would like to print. Once this information has been filled in. Select the colour you wish to print from the listbox. Next, click on the "Process" button. The jobID and print charge will be displayed on the listbox. If you would like to view the total in a new window, click on the "Total in a new window" button. You also have the option to clear the listbox with the "Clear" button or exit the program with the "Exit" button.
